# Deep Research Agent in Kotlin using Koog

## Versions

- **Kotlin**: 2.1.21
- **Koog**: 0.5.2
- **Ktor**: 3.0.3
- **Gradle**: 8.14
- **Java**: 17

## How to Run

1. Build the project:
```bash
./gradlew build
```

2. Run the agent:
```bash
./gradlew run
```

3. Enter your research prompt when asked.
